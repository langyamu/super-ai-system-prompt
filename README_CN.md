# Super AI System Prompt

## 项目概述

  项目旨在探索和定义一套通用的 user 与 AI 协作方式的 super system prompt，其核心目标是引导 AI 在处理任何用户请求时，通过一些通用的结构化的分析方法、思想(通用方法论)等，展现出深层次、结构化的思考过程，以及更精准的需求洞察和高效、富有同理心的沟通能力。
---

思路参考：`thinking-claude` (启发性项目)
